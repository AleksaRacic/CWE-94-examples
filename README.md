# CWE-94-examples

This repository contains examples of code that violates CWE-94: Improper Control of Generation of Code ('Code Injection')

## Running the examples

```npm install```

```node app.js```

If you want to use nodemon:

```npm install -g nodemon```

```nodemon app.js```

Landing page:

```http://localhost:3000/```
